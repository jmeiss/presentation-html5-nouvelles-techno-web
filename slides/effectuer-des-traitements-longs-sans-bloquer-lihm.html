Effectuer des traitements longs sans bloquer l'IHM

  • Web workers == JavaScript multi-threadé
    • Permet de lancer des tâches de fond en JavaScript
    • exemple
  • main.js 
          
var worker = new Worker('task.js');
worker.onmessage = function(event) { alert(event.data); };
worker.postMessage('data');
  • task.js 
          
self.onmessage = function(event) {
  // Do some long work
  self.postMessage("recv'd: " + event.data);
};
