Stockage local

Support : Chrome, FF, Opéra, Safari, IE

  • Stockage de clés-valeurs sur le navigateur 
          
localStorage.setItem("key", "value");
var value = localStorage.getItem("key");
    • Jusqu’à plusieurs mega-octets
    • Peut être stocké pendant un temps indéterminé tant que l’utilisateur n’a pas vidé son cache, ou seulement le temps d’une session utilisateur
  • Avantages :
    • Très simple à mettre en oeuvre
    • Attention : pas de sécurité si quelqu’un a accés au PC
    • Données peu structurées uniquement
