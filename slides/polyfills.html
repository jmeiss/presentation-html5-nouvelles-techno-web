Polyfills

  • polyfill (n) : librairie JavaScript qui réplique le comportement d'une API standard dans un navigateur ancien qui ne la supportait pas
  • Exemple : Socket.io
    • API unifiée pour la commication client <-> serveur
    • Utilise les Web Sockets sur les navigateurs récents, AJAX ou Flash sur les navigateurs anciens
  • ExplorerCanvas
    • Dessine des "Canvas" avec la technologie VML sur les anciennes versions d'Internet Explorer
